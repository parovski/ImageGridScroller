Task: Imgage Grid Scroller
Specifications
  • The images need to be displayed as tiles in a grid.
  • The width and height of the tiles are fixed, e.g. width:180px - height:250px .
  • The number of columns depends on the width of the screen.
  • The tiles need to have some spaces between them.
  • The tile itself shows the image and the title.
  • When hovering over the tile, it should show a tooltip with the description of the image.
  • The grid should allow to scroll vertically to the bottom of the list in a fluid, smooth way.
